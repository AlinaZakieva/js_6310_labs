# shoestore_website  
Разработка сайта магазина обуви с использованием JavaScript (React)

# Описание проекта

Проект состоит из двух частей:

- **`ui-library/`** — библиотека переиспользуемых React-компонентов для интернет-магазина обуви.
- **`frontend/`** — демонстрационное приложение ShoeStore, показывающее использование этих компонентов:
  каталог обуви, карточка товара и корзина.

### Компоненты библиотеки

1. **Button**
   - Базовая кнопка для действий.
   - Поддерживает разные размеры и состояния.
   - Используется в карточках товара, корзине и на странице товара.

2. **ProductCard**
   Карточка товара для каталога:
   - Фотография товара (обувь).
   - Название, описание и цена без копеек с символом `₽`.
   - Кнопка **«Добавить в корзину»**.
   - Адаптивная верстка.
   - Вся карточка кликабельна: по клику открывается страница карточки товара.

3. **ProductCatalog**
   Компонент каталога товаров:
   - Отображает сетку карточек обуви.
   - При клике по карточке вызывает обработчик для открытия страницы товара.
   - Передаёт колбэк для добавления товара в корзину.
   - Имеет пустое состояние (когда список товаров пуст).

# Демонстрационное приложение (frontend)

Приложение ShoeStore — мини-версия интернет-магазина обуви.

Реализованы три основных экрана:

1. **Каталог обуви**
   - Сетка карточек товаров на основе компонента `ProductCatalog`.
   - В каждой карточке:
     - фото обуви;
     - название модели;
     - короткое описание;
     - цена без копеек, `₽` не переносится на следующую строку;
     - кнопка **«Добавить в корзину»**.
   - Карточка товара кликабельна: при клике открывается отдельная страница товара.

2. **Карточка товара**
   - Слева — крупное фото выбранной модели обуви.
   - Справа:
     - название товара;
     - цена;
     - описание;
     - блок «Характеристики» (материал, размеры, производитель);
     - кнопка **«Добавить в корзину»**.
   - Есть кнопка **«← Назад в каталог»** для возврата к списку товаров.

3. **Корзина**
   - Значок корзины в правом верхнем углу шапки:
     - показывает количество товаров и итоговую сумму.
   - По клику открывается страница корзины:
     - если корзина пустая — текст **«Корзина пуста»**;
     - если есть товары — список позиций с названием и ценой;
     - для каждой позиции есть кнопка **«Удалить»**;
     - внизу отображается **итог: N шт · сумма ₽**.

Все данные о товарах сейчас хранятся в массиве `products` в `frontend/src/App.tsx`
(мок-данные без бэкенда).

## Технологии

- React 19 с TypeScript
- Vite — сборка и dev-сервер
- Jest + @testing-library/react — тестирование
- ESLint — линтинг кода (правила не ниже примерного проекта, 2 пробела, без `;`)
- CSS (общий `App.css` + стили библиотеки)

# Инструкция по запуску
Предварительные требования
	•	Node.js 18+
	•	npm
Установка и сборка UI Library:
    cd solutions/student-13/ui-library

    npm install
    npm run lint
    npm run test
    npm run test:coverage
    npm run build

	•	npm run lint — проверка кодстайла (ошибок и предупреждений быть не должно).
	•	npm run test — запуск тестов компонентов.
	•	npm run test:coverage — генерация отчёта покрытия (для одного из компонентов, например ProductCatalog, покрытие ≥ 90%).
	•	npm run build — сборка библиотеки и деклараций типов.

Запуск демо-приложения:
    cd ../frontend

    npm install
    npm run lint
    npm run test
    npm run build
    npm run dev
Приложение будет доступно по адресу: http://localhost:5173/
